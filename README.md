# results-f1-kotlin
 F1 Results App — Kotlin

Aplicación Android desarrollada en Kotlin que muestra los resultados de las carreras de Fórmula 1.
Incluye:

    Lista de resultados de Grandes Premios.

    Información básica de cada carrera.

    Opción para contactar con el desarrollador desde la app.

 Características principales

    Resultados de F1 con piloto, escudería y posición.

    Interfaz sencilla y rápida de navegar.

    Sección de Contacto para enviar un mensaje al desarrollador.

    Arquitectura clara y fácil de mantener.
